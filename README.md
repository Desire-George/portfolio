# 🛡️ Cybersecurity Portfolio

My personal cybersecurity portfolio website — built with pure HTML/CSS/JS, hosted on GitHub Pages.

## 🌐 Live Site
**https://DesireSec.github.io**

## 📁 Structure
```
index.html        ← The entire portfolio (single file)
README.md         ← This file
```

## 🚀 How to Deploy on GitHub Pages

1. Create a new GitHub repo named exactly: `yourusername.github.io`
   (replace `yourusername` with your actual GitHub username)

2. Upload `index.html` to the repo

3. Go to **Settings → Pages → Source → Deploy from branch → main**

4. Your site will be live at `https://yourusername.github.io` within 1-2 minutes

## ✏️ How to Customise

Open `index.html` in any text editor and search for:

| Placeholder | Replace with |
|---|---|
| `Your Name` | Your real name |
| `yourusername` | Your GitHub/LinkedIn/TryHackMe username |
| `yourname@email.com` | Your email address |
| `[Date]` | Actual dates |
| `[Blue Team / SOC Analysis...]` | Your actual focus area |
| `[Your Certification]` | Current cert you're pursuing |

## 📌 Sections
- **Hero** — Name, title, badges, terminal animation
- **About** — Bio, tools, skills, focus areas
- **Projects** — Home lab and hands-on work
- **Certifications** — Completed, in-progress, planned
- **CTF Write-ups** — Links to your GitHub write-up repos
- **Contact** — Email, LinkedIn, GitHub, TryHackMe
